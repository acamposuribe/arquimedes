# Completed Phase Docs

This folder archives plans and specs for finished phases.

- `plans/` keeps the implementation plans for completed work.
- `specs/` keeps the final design docs for completed work.

Active work should stay under:
- `docs/superpowers/plans/`
- `docs/superpowers/specs/`

Archived docs remain part of the project history and may still be referenced from `docs/PLAN.md`, `docs/PIPELINE.md`, and the global system spec.
